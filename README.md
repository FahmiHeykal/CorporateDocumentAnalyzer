# Corporate Document Analyzer

A Python-based application for analyzing corporate documents using natural language processing.

## Features

- Upload and analyze PDF/Word documents
- Extract text and metadata
- Generate executive summaries
- Identify action items, risks, opportunities, and decisions
- Sentiment analysis
- Document statistics
- Export results to PDF/Word

## Installation

1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Download spaCy model: `python -m spacy download en_core_web_sm`
4. Run the app: `streamlit run app.py`

## Usage

1. Upload a PDF or Word document
2. Select analysis mode
3. View results in the main panel
4. Export results if needed
